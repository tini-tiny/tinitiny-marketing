# MATRIX — Sistema de Produção de Conteúdo
## Tini Tiny | Documento para Agente de IA

> **LEIA ANTES DE USAR:** Consulte brand.md + ANGLES.md + HOOKS.md + VOICE.md antes de gerar qualquer peça.
> A MATRIX é o sistema operacional — combina os ângulos com os hooks e transforma em execução semanal.

---

## FLUXO DO AGENTE

```
1. Receber slot (ex: "TikTok segunda, estampa Whimsical")
2. Consultar MATRIX → identificar ângulo + hook do slot
3. Consultar ANGLES.md → receita visual completa
4. Consultar HOOKS.md → hook exato + variações
5. Consultar VOICE.md → tom e linguagem
6. Usar PROMPT PADRÃO com parâmetros preenchidos
7. Checar CHECKLIST PRÉ-PUBLICAÇÃO
8. Apresentar para aprovação da Cami
9. Publicar após aprovação
```

---

## CADÊNCIA SEMANAL ALVO

| Canal | Frequência | Formato | Observação |
|---|---|---|---|
| TikTok | 7x/semana (1x/dia) | Vídeo MP4 1080×1920 | Prioridade de crescimento |
| Instagram Reels | 4–5x/semana | Derivado do TikTok | Republicar com ajuste de legenda |
| Instagram Feed | 3–4x/semana | Foto 1:1/4:5 ou carrossel | Editorial, curado |
| Instagram Stories | Diário | Variado | Humanização + engajamento |
| Pinterest | 10+ pins/semana | 1000×1500px | Alto volume, automático |

## SPECS DE OUTPUT POR PLATAFORMA

### TikTok
```
Dimensão: 1080×1920px (9:16)
Duração: 15–45s (ideal 20–30s)
Frame rate: 24fps ou 30fps
Codec: H.264 MP4
Áudio: SEM MÚSICA (adicionar nativamente no TikTok após upload)
Texto: Poppins Bold, #FFFFFF ou #C1D9D7 (sage)
Fonte: mínimo 52pt (legível em tela de 6")
Posição do texto: zona segura — 15% das bordas
Ritmo: mínimo 3.5s por frame de texto, texto permanece 4s+
Assinatura final: "tini tiny | @tini.tiny.shop" — Poppins Bold, 28pt, sage
Cores no entorno: NUNCA cor primária saturada
```

### Instagram Reels
```
Dimensão: 1080×1920px (9:16)
Duração: 15–30s
Codec: H.264 MP4
Áudio: pode ter música integrada (diferente do TikTok)
Thumbnail: extrair frame mais bonito — produto em overhead editorial
Specs de texto: igual TikTok
```

### Instagram Feed — Foto
```
Dimensão: 1080×1080px (1:1) ou 1080×1350px (4:5)
Formato: JPG, máximo 10MB
Paleta: sage #C1D9D7 / off-white #FFFFFF / madeira natural — NUNCA primárias
Ângulo padrão: overhead 90°
Luz: natural, high-key, difusa
Props: Maileg, Little Saw, Konges Slojd, livros Beatrix Potter — nada plástico
```

### Instagram Feed — Carrossel
```
Dimensão: 1080×1080px cada slide
Slides: mínimo 3, máximo 7
Slide 1: hook visual mais forte (thumbnail que para o scroll)
Slides 2–N: desenvolvimento, detalhes, prova
Último slide: CTA + logo tini tiny
```

### Instagram Stories
```
Dimensão: 1080×1920px
Zona segura: 250px do topo, 400px da base (para stickers e username)
Paleta: sage #C1D9D7 como cor de destaque
Duração foto: 5–7s | Duração vídeo: até 60s
```

### Pinterest
```
Dimensão: 1000×1500px (2:3) — OBRIGATÓRIO
Formato: JPG ou PNG
Máximo 10MB
Texto sobreposto: Poppins Bold, máximo 1 headline
Boards disponíveis (destino de cada pin):
  - "Tapetes & Quartos" → ambientes completos com tapete
  - "Estampas Exclusivas" → close no produto, detalhes
  - "Cool Moms Club" → lifestyle, identidade, mãe + filho
  - "Unboxing & Presentinhos" → embalagem, gift, sacola sage
  - "Inspo de Quarto" → inspiração geral (não só Tini Tiny)
  - "Por que Tini Tiny" → contraste, tecnologia, segurança
  - "Desenvolvimento & Segurança" → bebê em desenvolvimento, A07/A13
```

## CHECKLIST PRÉ-PUBLICAÇÃO

### Visual
- [ ] Sem rosto frontal de criança em vídeo
- [ ] Sem brinquedo plástico ou cenário stock
- [ ] Cores no entorno: neutras (sage, off-white, madeira) — sem primárias
- [ ] Luz natural, high-key
- [ ] Assinatura "tini tiny | @tini.tiny.shop" no último frame
- [ ] Poppins Bold, mínimo 52pt, legível em mobile

### Copy / Legenda
- [ ] Hook na primeira linha — sem saudação ou nome da marca
- [ ] Sem menção a desconto ou preço (exceto slots de urgência autorizados)
- [ ] Tom cool mom friend — não corporativo, não gritante
- [ ] CTA criativo da biblioteca (não "link na bio" isolado)
- [ ] Hashtags do set principal incluídas
- [ ] "Cool Moms Club" ✅ | "clube das mães" ❌
- [ ] "motherhood is not boring" ✅ | "maternidade não é chata" ❌
- [ ] "tiragem limitada" ✅ | "últimas chapas de EVA" ❌
- [ ] "pano úmido com sabão neutro" ✅ | "esfregar" ❌
- [ ] Sem revelar processo técnico (patente pendente)
- [ ] Produção = "30–40 dias, feito a mão, made to order" ✅

### Técnico
- [ ] Dimensão correta para a plataforma
- [ ] Vídeo sem música embutida (TikTok)
- [ ] Ritmo lento verificado (3.5s+ por frame)
- [ ] Pinterest: 1000×1500px, board correto preenchido
